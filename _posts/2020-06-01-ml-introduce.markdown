## 人工智能发展历程
    
| | 推理期<br>(1950s 至 pre-1970s) | 知识期<br>(mid-1970s 至 1980s) | 学习期<br>(1980s 至今 ) |
|-|--------------------------------|------------------------------|-----------------------|
| 观点 | 智能 = 逻辑推理能力 | 使机器拥有知识 | 使机器自己学习知识 <br> ↑ 机器学习 |
| 应用 | 证明数学定理 | 大量专家系统问世 | 计算机视觉、自然语言处理等“计算机应用”领域 |
| 缺陷 | 仅有逻辑推理能力无法实现人工智能 | 知识工程瓶颈 |


## 机器学习的划分

* 从样例中学习  -- **归纳学习**
* 在问题求解何规划中学习
* 通过观察和发现学习 -- 类比学习
* 从指令中学习 -- 示例学习


## 归纳学习的演进

#### 1. 符号主义学习(1980s)，产生明确的概念表示

* 代表：
    * 决策树学习：模拟人类判别概念的树形流程。
    * 基于逻辑的学习：其代表是归纳逻辑程序设计( ILP )，用谓词逻辑表示知识。
* 局限：知识表示能力很强 -> 假设空间太大 -> 问题规模受限

#### 2. 连接主义学习 (mid-1990s)，产生黑箱模型

* 代表：BP 算法，基于神经网络。
* 局限："试错性"，参数设置缺乏理论指导。

#### 3. 统计学习( mid-1990s 至今)，以统计学习理论为直接支撑

* 代表：支持向量机( SVM ) 和更一般的核方法( kernel methods )。

#### 4. 连接主义学习( 2000s 至今 )

* 代表：深度学习，即很多层的神经网络，模型复杂度非常高。
* 依赖：大数据，以免过拟合；强力计算设备。
* 局限：理论创新不明显；应用范围有限。


## 数据挖掘与机器学习的联系


* 数据挖掘：从海量数据中发掘知识
    - 数据库：提供数据管理 
    - 机器学习：提供数据分析
        - 统计学
